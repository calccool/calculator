FoV calculator: Viewing angle conversion between CS2, Valorant, Apex Legends and other shooters

Consistency in muscle memory is critical to achieving great results in competitive shooters. Field of View directly affects the visual speed of objects and the perceived sensitivity of the mouse. If you are playing CS2 with FoV 90 and go to Valorant, simply copying numbers will not work, since these games use different mathematical models for calculating space.

Our calculator allows you to instantly synchronize settings between different engines so that the image scale and “feel” of the game remain identical.

How different games measure viewing angle

There are three main standards that modern developers use:

  • Horizontal 4:3 (Source Engine): A classic for games like CS:GO, CS2, Apex Legends and Team Fortress 2. The base value is always calculated using a 4:3 aspect ratio, even if you have a widescreen monitor.
  • Horizontal 16:9: The standard for modern titles such as Valorant, Overwatch 2 and PUBG. Here the viewing angle is tied to a standard Full HD monitor.
  • Vertical FoV: Used in Rainbow Six Siege, Battlefield, Escape from Tarkov and games on the Unity engine. The angle is measured along the vertical axis, and the horizontal view expands automatically based on the width of your screen.

Mathematical calculation model

For accurate translation, we use trigonometric formulas relating vertical angle (VFOV), horizontal angle (HFOV) and monitor aspect ratio (Aspect Ratio).

Basic conversion formulas

  1. Finding vertical FoV from horizontal: $$VFOV = 2 \cdot \arctan\left(\frac{\tan(HFOV_{deg} \cdot \frac{\pi}{360})}{AR}\right) \cdot \frac{180}{\pi}$$ Where $AR$ is the aspect ratio (for example, $1.333$ for 4:3 or $1.777$ for 16:9).
  2. Finding horizontal FoV from vertical: $$HFOV = 2 \cdot \arctan\left(\tan(VFOV_{deg} \cdot \frac{\pi}{360}) \cdot AR\right) \cdot \frac{180}{\pi}$$
  3. Calculation of true horizontal view: It is this parameter that shows how much gaming space you actually see on your monitor. The calculator automatically calculates it based on the selected aspect ratio (16:9, 21:9, etc.).

How to use the calculator

To get the ideal settings, follow these steps:

  1. Select source engine: Specify the game from which you want to transfer settings.
  2. Enter current FoV: The value you currently have. Use quick entry buttons for popular values ​​(90, 103, 110).
  3. Select target engine: The game you are about to play.
  4. Please indicate your monitor: This is critical to the calculation True horizontal view. If you have a standard monitor, choose 16:9. For ultra-wide panels - 21:9.
  5. Click "Calculate": You will receive a ready-made number to insert into the game settings, and will also see a visualization of coverage on the Visibility Radar.

Why is Visibility Radar important?

Our calculator is the only tool that visualizes the difference between vertical and horizontal views in real time. The blue cone shows how much the image is “cut off” or expanded in height, and the green cone shows the actual coverage on the sides. This helps to determine if the fisheye effect will be too fisheye at high FoV values ​​on widescreen monitors.
