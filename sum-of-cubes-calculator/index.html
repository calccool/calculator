What is the sum of cubes?

The sum of cubes is the result of adding numbers raised to the third power. This mathematical operation is often found in algebra, calculus, and number theory. Our tool allows you to instantly calculate the sum for a sequence of the first $n$ natural numbers $(1^3 + 2^3 + … + n^3)$ or for any arbitrary range, including even and odd steps.

Basic formula

To find the sum of cubes of the first $n$ consecutive natural numbers, use the following formula:

$$S_n = \left( \frac{n(n + 1)}{2} \right)^2$$

Where:

  • n — the number of consecutive natural numbers (upper bound).
  • S_n - total amount.

This formula is notable for the fact that the sum of the cubes of numbers is equal to square of the sum the same numbers.

The Amazing Connection: Nicomachus' Theorem

There is a beautiful relationship in mathematics known as Nicomachus' Theorem. It states that the sum of the cubes of the first $n$ integers is equal to the square of the $n$ th triangular number.

Simply put:

$$1^3 + 2^3 + 3^3 + … + n^3 = (1 + 2 + 3 + … + n)^2$$

Example for n = 3:

  • Sum of numbers: $1 + 2 + 3 = 6$
  • Sum Square: $6^2 = 36$
  • Sum of cubes: $1 + 8 + 27 = 36$

Our calculator automatically checks this property when calculating sequences with step 1.

Manual calculation example

Let's say we need to find the sum of cubes from 1 to 4.

  1. Raise each number to the power of 3:
    • $1^3 = 1$
    • $2^3 = 8$
    • $3^3 = 27$
    • $4^3 = 64$
  2. We add up the results: $1 + 8 + 27 + 64 = 100$ .

Or use the formula:

$S_4 = (4 \times 5 / 2)^2 = (20 / 2)^2 = 10^2 = 100$.

The results are the same. But why count manually if you can use the form above?
