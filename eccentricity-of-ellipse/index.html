What is eccentricity?

In geometry, eccentricity is a parameter characterizing the “flattenedness” of a conical section. For an ellipse this value is always in the range from 0 to 1.

  • If $e = 0$, then the figure is circumference. The foci coincide with the center, and the semi-axes are equal.
  • If $0 < e < 1$, then the figure is ellipse. The closer the value is to one, the more elongated the ellipse is.
  • If $e \rightarrow 1$, the ellipse degenerates into a segment (straight line).

Basic formulas (Formulas)

The calculation is based on the ratio of the major ( $a$ ) and minor ( $b$ ) semi-axes.

1. Linear eccentricity (Focal distance):

First, the distance from the center of the ellipse to its focus ( $c$ ) is calculated:

$$c = \sqrt{a^2 — b^2}$$

2. Eccentricity:

This is the ratio of the focal length to the semimajor axis length:

$$e = \frac{c}{a}$$

Or through the direct formula for the dependence of the axes:

$$e = \sqrt{1 — \frac{b^2}{a^2}}$$

Practical Application

Understanding eccentricity is important not only in school geometry, but also in real science:

  • Astronomy: Kepler's first lecture states that the orbits of the planets are ellipses. For example, the eccentricity of the Earth's orbit is very small ( $\approx 0.0167$ ), so it is almost circular, but Halley's comet has a huge eccentricity ( $\approx 0.967$ ), so it flies far from the Sun.
  • Engineering: Calculation of loads on arched structures and mechanism parts (cam shafts).
  • Optics: Design of elliptical reflectors.
