What is the FOIL method?

FOIL is a mnemonic rule (acronym) from the English language that helps you remember the order in which terms of two binomials are multiplied. It ensures that you multiply every term in the first bracket by every term in the second.

The abbreviation stands for as follows:

  • F irst ( First ): Multiply the first terms in each bracket.
  • O uter ( External ): Multiply the outer terms (extreme).
  • I nner ( Domestic ): Multiply the internal terms (average).
  • L ast ( Latest ): Multiply the last terms in each bracket.

Algebraic formula

For expression

$$(ax + b)(cx + d)$$

The disclosure formula looks like this:

$$Result = \underbrace{acx^2}_{First} + \underbrace{adx}_{Outer} + \underbrace{bcx}_{Inner} + \underbrace{bd}_{Last}$$

After bringing similar terms (

$$adx$$

And

$$bcx$$

) we get the standard quadratic equation:

$$y = Ax^2 + Bx + C$$

Common mistakes in manual calculations

  • Loss of characters: The most common mistake is ignoring the minus sign before a number (for example, multiplying by 5 instead of -5).
  • Forgotten terms: Multiplying only the first and last terms (a common beginner mistake), skipping the cross multiplication.
  • Addition errors: Incorrect casting of similar terms (for example, addition of $$x^2$$ and $$x$$ ).
