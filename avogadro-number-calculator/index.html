Avogadro's number calculator: Calculation using crystal lattice parameters

Avogadro's number ( $N_A$ ) is one of the fundamental physical constants that connects the microworld of atoms with the macroworld that we see. We usually take this number for granted ( $6.022 \times 10^{23}$ ), but how were scientists able to calculate it?

Our online calculator allows us to repeat this historical experiment using X-ray diffraction method. Knowing the density of a substance, its molar mass and the structure of the crystal lattice, it is possible to calculate with high accuracy the number of atoms in one mole.

Calculation formula

The calculations are based on the relationship between the macroscopic density of a substance and the density of its unit cell. If we assume that the crystal is ideal, these densities are equal.

The equation for finding Avogadro's number is as follows:

$$N_A = \frac{Z \times M}{\rho \times V}$$

Where:

  • $N_A$ — Avogadro’s number (the desired value, mol⁻¹).
  • $Z$ — The number of formula units (atoms or molecules) in one unit cell.
  • $M$ — Molar mass of the substance (g/mol).
  • $\rho$ (ro) - Density of the substance (g/cm³).
  • $V$ — Unit cell volume (cm³).

How to find the cell volume ( $V$ )?

In laboratory problems, it is often not the volume that is given, but lattice parameter (edge ​​length $a$ ). For the cubic system, the volume is calculated simply:

$$V = a^3$$

Important: Lattice parameter is usually measured in Angstroms ( $\mathring{A}$ ), nanometers ( $nm$ ) or picometers ( $pm$ ). Our calculator automatically converts these values ​​into centimeters for correct calculation.

Crystal lattice parameters

To get an accurate result, it is important to correctly specify the number of particles ( $Z$ ) in the unit cell. This depends on the type of Bravais lattice.

Calculation example (Copper)

Let's test the calculator using copper ( $Cu$ ), which has a face-centered cubic (FCC) lattice.

  1. $Z$ number: For fcc lattice $Z = 4$ .
  2. Molar mass ( $M$ ): $63.546$ g/mol.
  3. Density ( $\rho$ ): $8.96$ g/cm³.
  4. Lattice parameter ( $a$ ): $3.615 \mathring{A}$ (Angstrom).

Substituting these values ​​into the formula:

  1. Convert $a$ to cm: $3.615 \times 10^{-8}$ cm.
  2. We calculate the volume of $V = (3.615 \times 10^{-8})^3 \approx 4.72 \times 10^{-23}$ cm³.
  3. We consider $N_A = (4 \times 63.546) / (8.96 \times 4.72 \times 10^{-23})$ .

The result will be very close to the table value $6.02 \times 10^{23}$.

Why count the error?

In real laboratory work, the obtained value may differ from the CODATA standard ( $6.02214076 \times 10^{23}$ ). This is normal and may be caused by:

  • Defects of a real crystal (vacancies, dislocations).
  • The presence of isotopes that affect the average molar mass.
  • Density measurement error.

Our tool automatically calculates relative error as a percentage, which will help you when drawing up conclusions for a scientific paper.
