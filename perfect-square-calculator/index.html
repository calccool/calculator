What is a perfect square?

In mathematics perfect (or perfect) square is a number that can be represented as the product of some integer by itself. In other words, this is a number from which the square root is taken entirely, without a remainder.

The formula for a perfect square looks like this:

$$n = a^2$$

Where:

  • $n$ - the number to be checked (for example, 9).
  • $a$ is an integer (in this case 3, since $3 \times 3 = 9$ ).

The geometric meaning of this concept is simple: if you have $n$ of identical small squares, you can add them into one large solid square only if $n$ is a perfect square.

Properties and signs of a complete square

If you are solving a problem without a calculator, it is useful to know a few mathematical properties that will help you quickly weed out numbers that are definitely not squares:

  • Last digit: A perfect square can only end in numbers 0, 1, 4, 5, 6 or 9. If a number ends in 2, 3, 7 or 8, it is guaranteed not to be a perfect square.
  • Digital root: The sum of the digits of a perfect square (reduced to one digit) is always 1, 4, 7 or 9.
  • Parity of zeros: If a number ends in zeros, the number must be even (for example, 100, 2500, 10000). The number 1000 (three zeros) is not a perfect square.
