The mathematical model of name resonance ( $R_{names}$ ) is as follows:

$$R_{names} = \left( \sum_{i=1}^{n} Code(Char_A) + \sum_{j=1}^{m} Code(Char_B) \right) \pmod{100}$$

Where:

  • $Char_A$ and $Char_B$ are partner name symbols.
  • $Code$ is the unique numerological code of the letter.
  • $\pmod{100}$ - result normalization module.
