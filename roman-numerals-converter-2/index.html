Mathematical rules for forming numbers

Roman notation is based on two simple principles: additive (addition) and subtractive (subtraction).

Addition principle

If a digit with a lesser or equal value is after (to the right) of the digit with a larger value, their values ​​are added together.

Formula:

$$N = V_1 + V_2 + … + V_n$$

Examples:

  • VI = $5 + 1 = 6$
  • XV = $10 + 5 = 15$
  • MMXXV = $1000 + 1000 + 10 + 10 + 5 = 2025$

Subtraction principle

If the number with the lower value is before (to the left) of the number with a larger value, the smaller one is subtracted from the larger one. This rule is used to avoid repeating one character four times (for example, instead of IIII they write IV).

Formula:

$$N = V_{большая} — V_{меньшая}$$

Subtraction rules:

  1. I can only be deducted from V And X ($4, 9$).
  2. X can only be deducted from L And C ($40, 90$).
  3. C can only be deducted from D And M ($400, 900$).
  4. Symbols V, L, D are never deducted.

Examples:

  • IV = $5 — 1 = 4$
  • IX = $10 — 1 = 9$
  • XL = $50 — 10 = 40$
  • CM = $1000 — 100 = 900$

Limitations of the classical system

Classical Roman numbering has a number of features that our algorithm takes into account:

  1. Missing zero: The Romans did not have a symbol for the number 0 (they used the word nulla - nothing).
  2. Maximum number: In standard notation (without using the vinculum overscore), the maximum number is 3999 (MMMCMXCIX).
  3. Repetitions: One symbol cannot be repeated more than three times in a row (except in rare historical variations on watch dials, where 4 is designated as IIII). Our validator automatically checks the correctness of your input and will tell you if you made a mistake in the syntax.
