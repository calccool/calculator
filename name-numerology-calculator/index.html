How the calculation works: Formulas and algorithms

The calculator is based on the classical Pythagorean system, where each letter corresponds to a number from 1 to 9. The calculation process takes place in several stages, including the “numerological convolution” method.

Mathematical calculation model

To obtain the final value, use the formula for the sum of the ordinal values ​​of the letters:

$$S = \sum_{i=1}^{n} V_i$$

Where:

  • $S$ - total amount of vibrations;
  • $V$ - the numeric value of a specific letter;
  • $n$ - number of letters in the name.

After receiving the amount $S$ , if it is not Master number (11, 22 or 33), it is reduced (added) to a single digit number. For example, if $S = 42$ , then the resulting value will be $4 + 2 = 6$ .
