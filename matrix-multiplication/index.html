Matrix multiplication is one of the fundamental operations in linear algebra, which is widely used in computer graphics, physics, machine learning and economics. Compatibility condition for multiplication

The main rule that must be followed to calculate the product $C = A \times B$ is the following:

The number of columns in the first matrix $A$ must be equal to the number of rows in the second matrix $B$ .

If the $A$ matrix is ​​of size $m \times k$ and the $B$ matrix is ​​of size $k \times n$ , then the resulting $C$ matrix will be of size $m \times n$ .

If the condition $k = k$ not performed, multiplication impossible. Our calculator will automatically check this and notify you.

Matrix multiplication formula

Each element $c_{ij}$ of the resulting matrix $C$ is the sum of the products of the elements $i$ of the th row of the matrix $A$ by the corresponding elements of the $j$ th column of the matrix $B$ .

Formula for finding the element $c_{ij}$ :

$$c_{ij} = \sum_{l=1}^{k} a_{il} b_{lj}$$

Where:

  • $c_{ij}$ is an element of the $C$ matrix at the intersection of the $i$ th row and the $j$ th column.
  • $a_{il}$ — elements of the $i$ th row of the $A$ matrix.
  • $b_{lj}$ — elements of the $j$ th column of the $B$ matrix.
  • $k$ - number of $A$ columns (and $B$ rows).
