Mathematical calculation model

To ensure 100% accuracy, the calculator converts each input value into a single unit of measurement - seconds, sums them up and performs the reverse conversion.

The general timing formula is as follows:

$$T_{total} = \sum_{i=1}^{n} (H_i \times 3600 + M_i \times 60 + S_i)$$

Where:

  • $T_{total}$ — final time in seconds;
  • $H, M, S$ - hours, minutes and seconds of each individual track;
  • $n$ - total number of added positions.

If limit mode is enabled, the difference ( $\Delta$ ) is calculated:

$$\Delta = L — T_{total}$$

Where $L$ is the specified airtime limit. A positive $\Delta$ value means there is free space, a negative value means overflow (overtime).

How to use the calculator

To get an accurate result, follow a simple algorithm:

  1. Enter the duration values ​​for each track in the text field. Each new value must be new line.
  2. If you are planning an ad unit, use the “+15 sec” or “+30 sec” quick entry buttons.
  3. If you have a strict time limit (for example, an hour slot), activate the switch “Take into account the airtime limit” and enter the valid time.
  4. Click the button "Calculate".
  5. Examine the summary block: the system will show the total number of tracks, total timing and remaining (or overrun) time.
