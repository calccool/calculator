How the GPA calculator works

The program is based on the classic principle of arithmetic mean and performs all calculations instantly.
Calculation formula:

GPA = (5 × A + 4 × B + 3 × C + 2 × D) / (A + B + C + D)

Where:

  • A — number of ratings “5”;
  • B — number of ratings “4”;
  • C — number of ratings “3”;
  • D — number of ratings “2”.

Example:
If a student has 3 A’s, 4 B’s and 3 C’s, then:
(5×3 + 4×4 + 3×3) / (3 + 4 + 3) = (15 + 16 + 9) / 10 = 4.0 - the final grade is “four”.
