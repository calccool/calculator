What is a lucky number?

A number is considered lucky if successive replacement of the number by the sum of the squares of its digits ultimately leads to one. If the process goes in cycles and never reaches one, this number is called “unhappy” or “sad” (Sad Number).

The verification algorithm looks like this:

  1. Take the natural number $$n$$ .
  2. Let's break it down into numbers.
  3. We square each number and sum them up.
  4. We repeat the process for the received amount.

Mathematically this process for number

$$n$$

with numbers

$$d_1, d_2, …, d_k$$

can be written by the formula:

$$S(n) = \sum_{i=1}^{k} d_i^2$$

The process continues until

$$S(n) = 1$$

(happy outcome) or until the sequence enters an infinite loop (unfortunate outcome).

Example calculation

Consider the number 19. Is it happy?

  1. $$1^2 + 9^2 = 1 + 81 = 82$$
  2. $$8^2 + 2^2 = 64 + 4 = 68$$
  3. $$6^2 + 8^2 = 36 + 64 = 100$$
  4. $$1^2 + 0^2 + 0^2 = 1$$

The result is 1, therefore 19 is a lucky number.

Unlucky numbers and the "Sad Cycle"

If a number is not lucky, the sum of the squares of its digits will never equal one. Instead, the sequence ends up in a unique loop that repeats itself endlessly. In the decimal number system, this cycle always consists of the same 8 numbers:

$$4 \to 16 \to 37 \to 58 \to 89 \to 145 \to 42 \to 20 \to 4 …$$

If in the process of calculations you come across any of these numbers (for example, 4 or 89), you can immediately say that the original number is unlucky.
