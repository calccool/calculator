Mathematics and processing algorithms

During the conversion process, we use interpolation and color correction algorithms. One of the key functions is to calculate the final file resolution depending on the selected scaling factor.

The final resolution is calculated using the formula:

$$R_{out} = (W_{orig} \cdot S) \times (H_{orig} \cdot S)$$

Where:

  • $W_{orig}$ — original matrix width;
  • $H_{orig}$ — initial matrix height;
  • $S$ — scale factor, where $0.1 \le S \le 2.0$ .

When applying brightness and contrast filters, the values ​​of each $P$ pixel are recalculated according to a linear function:

$$P_{new} = \text{clamp}((P_{old} \cdot \text{Contrast}) + \text{Brightness})$$

Why is PNG better for output from DNG?

The PNG format supports transparency and uses the Deflate compression algorithm, which does not introduce visual artifacts, unlike JPEG. This makes it an ideal choice for preserving maximum detail obtained from a RAW source.
