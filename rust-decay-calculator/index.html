Rust Base Decay Calculator: Decay Time Calculation

Mechanics Decay (rotting) in Rust, this is the process of gradual destruction of buildings, which is activated if there are not enough resources in the cabinet (Tool Cupboard) to contain it. This is a critical aspect of survival: if you forget to replenish your closet or your base is partially overrun, the building will disappear, opening up access to loot for everyone.

Our calculator allows you to instantly calculate two scenarios:

  1. Full cycle: How long will the base last with the current amount of resources + time for the destruction of the blocks itself.
  2. Pure Rot: How much time is left for a specific block (wall, foundation) if it is already damaged or the cabinet is empty.

Complete disintegration time based on materials

Each type of material has its own “resistance” to rotting. The more expensive the material, the longer it takes to break down after maintenance resources have run out.

Mathematical formula for calculation

The calculator uses algorithms that take into account the linear dependence of strength on time.

1. Calculation of maintenance time (Upkeep Time):

If there are resources in the closet, the safe time for the base is calculated by the formula:

$$T_{safe} = \frac{R_{stored}}{R_{daily}} \times 24$$

Where:

  • $R_{stored}$ — number of resources in the cabinet;
  • $R_{daily}$ - daily consumption (indicated in the cabinet interface).

2. Calculation of decay time (Decay Time):

If a block is damaged, the time until it disappears is reduced proportionally:

$$T_{decay} = \frac{HP_{current}}{HP_{max}} \times T_{max\_decay}$$

Where:

  • $HP_{current}$ - current health of the block;
  • $T_{max\_decay}$ - decay time constant for a given material (from 1 to 12 hours).

Main reasons for accelerated Decay

  • Lack of cabinet: If a cabinet is destroyed during a raid, rotting begins immediately.
  • Lack of resources: If there is only stone in the cabinet, and part of the base is built of metal, the metal fragments will begin to rot immediately.
  • External walls: Tall stone and wood walls rot much faster if they are not within the range of the cabinet.
  • Multi-storey buildings: The more blocks in the radius of the cabinet, the higher the percentage of Upkeep tax.

How to use the calculator

To get an accurate forecast, follow these steps:

  1. Select mode: “Full cycle” if you want to find out the time taking into account the resources in the closet, or “Rotting only” if you need to calculate the life time of a separate wall.
  2. Specify material: Select the building type (from thatch to armor).
  3. Enter data from the game:
    • Amount of resources (wood, stones, metal fragments or MVK);
    • Daily content value from the cabinet menu.
  4. Check out the visualization: An interactive graph will show the relationship between safe time and the final destruction process.
