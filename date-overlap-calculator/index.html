Date intersection calculator: checking time overlap online

When working with schedules, contracts, or vacations, you often need to quickly determine whether two time periods overlap. Our date crossing calculator is a highly accurate tool that not only gives a “yes” or “no” answer, but also provides a detailed mathematical calculation of each period.

The tool uses an "inclusive counting" algorithm, which is critical for legal and accounting accuracy, where the first and last days of the period are considered full-fledged work units.

Main features of the calculator

Using this service allows you to automate the verification of complex charts. Key features include:

  • Automatic overlay detection: The system instantly calculates the total days between two intervals.
  • Timeline visualization: An interactive Gantt chart clearly shows the position of periods relative to each other.
  • Intersection share calculation: You will find out what percentage of time one period occupies in the total time frame.
  • Error correction: If the start date is entered later than the end date, the calculator will automatically swap them for correct calculation.

Mathematical calculation model

The logic of the calculator is based on comparing the extremes of two numerical sets (time stamps). The following formulas are used to calculate the intersection interval:

Let $S_1, E_1$ be the beginning and end of the first period, and $S_2, E_2$ the beginning and end of the second.

  1. Start of intersection: $$S_{overlap} = \max(S_1, S_2)$$
  2. End of intersection: $$E_{overlap} = \min(E_1, E_2)$$
  3. Duration of application in days: If $S_{overlap} \le E_{overlap}$ , then: $$D = (E_{overlap} — S_{overlap}) + 1$$ Otherwise, there is no intersection and the gap is calculated.
