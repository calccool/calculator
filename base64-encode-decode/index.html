Base64 Online: Two-way String Converter (Encode/Decode) with URL-Safe support

Base64 is a standard for converting binary data into a string of 64 printable ASCII characters. This method is invaluable when you need to transfer data (for example, images or complex keys) through protocols that only handle text characters correctly.

Our calculator performs the conversion using an algorithm that splits every 3 bytes of input data into 4 blocks of 6 bits. Each such block corresponds to a specific character from the Base64 alphabet.

Mathematical model of data volume

An important feature of Base64 is the increase in the size of the resulting string. Since every 3 bytes turns into 4 characters, the volume increases by approximately 33.3%. The exact length of the output string $L_{out}$ depending on the length of the input string $L_{in}$ is calculated by the formula:

$$L_{out} = 4 \times \lceil \frac{L_{in}}{3} \rceil$$

Where $\lceil \dots \rceil$ is a function to round up to the nearest integer.

Instructions for using the tool

How to encode a string in Base64?

  1. Select the "Encode to Base64" option at the top of the calculator.
  2. Paste the source text into the Source Text field.
  3. If necessary, activate the “URL-safe format” switch if you plan to use the result in links.
  4. Click the button "Calculate".
  5. Copy the resulting hash using the quick copy icon.

How to decode Base64 back to text?

  1. Switch the tab to “Decode from Base64” mode.
  2. Paste the encrypted sequence of characters.
  3. If the string has no characters=Finally, make sure that URL-safe mode is enabled - the calculator itself will restore the necessary alignment.
  4. Click "Calculate" and get the original readable string.
