CHMOD calculator online: Linux permissions generator (755, 644, 777)

In operating systems of the Linux and Unix family, file security management is based on a system of access rights. Teamchmod(change mode) is used to change the access mode to file system objects. Each file or directory has three types of rights for three categories of users.

User categories

  • Owner (User): The file creator or current owner.
  • Group: Users who are members of the group to which the file is assigned.
  • Others: All other users in the system.

Basic access rights

  1. Read (Read, r): Allows you to view the contents of a file or a list of files in a directory.
  2. Write (w): Gives the right to change the contents of a file or create/delete files in a folder.
  3. Execute (x): Allows you to run a file as a program/script or enter a directory.

Mathematical model for calculating rights (Octal Notation)

The digital format of access rights is based on the octal number system. Each right has a certain “weight”, and the final value for each category of users is obtained by adding them up.

The calculation for one category is described by the formula:

$$P = (R \cdot 4) + (W \cdot 2) + (X \cdot 1)$$

Where:

  • $R$ - read bit (1 if active, 0 if not);
  • $W$ - write bit (1 or 0);
  • $X$ - execution bit (1 or 0).

Bit value table

Therefore, if you need to give read (4) and execute (1) permissions, the resulting number would be $4 + 1 = 5$ . The maximum value for one category is 7 ($4+2+1$).

Special rights: SUID, SGID and Sticky Bit

Our calculator supports "Advanced Mode" which includes special access flags. These bits are added by the fourth (leading) digit in octal code.

  • SUID (Set User ID): When installed on an executable file, the program runs with the rights of the owner of the file, and not the user who called it.
  • SGID (Set Group ID): Similar to SUID, but the process gains group rights. For directories, this means that new files inside will inherit the group of that directory.
  • Sticky Bit: In modern systems it is used for directories (for example,/tmp) to allow files to be deleted only by their owners, even if others have write access to the folder.

Formula for the full four-digit code:

$$Mode = (S_{bit} \cdot 1000) + (U \cdot 100) + (G \cdot 10) + O$$

Where $S_{bit}$ is the sum of special flags (SUID=4, SGID=2, Sticky=1).
