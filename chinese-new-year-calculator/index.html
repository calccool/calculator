Formula for identifying an animal and an element

For those who prefer manual calculations to software algorithms, there are mathematical models for determining the sign.

Animal index calculation

To find out which animal rules the year, use the remainder formula:

$$Index_{animal} = (Year — 4) \pmod{12}$$

Where the result corresponds to the serial number in the cycle (0 - Rat, 1 - Bull... 11 - Pig). In our calculator, this calculation is automated and takes into account the system shift.

Definition of element

The elements change every two years. The formula for determining the index of an element looks like this:

$$Index_{element} = \lfloor \frac{(Year — 4) \pmod{10}}{2} \rfloor$$

The resulting number from 0 to 4 indicates the corresponding element in the Wu-sin sequence.
