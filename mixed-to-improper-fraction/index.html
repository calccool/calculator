How to manually convert a mixed fraction to an improper fraction?

Although our calculator does this instantly, understanding the process is very important. The conversion is done in three simple steps based on one simple formula.

Conversion formula

To convert a mixed fraction of the form $A \frac{B}{C}$ (where $A$ is the integer part, $B$ is the numerator, and $C$ is the denominator) to an improper fraction $\frac{N}{C}$ , use the following formula to find the new numerator $N$ :

$$N = (A \times C) + B$$

Important: The denominator of $C$ in the new improper fraction remains the same as in the original fraction.

Step-by-step analysis with examples

Let's look at the process using two illustrative examples.

Example 1: Convert $2 \frac{1}{3}$

  1. We define the components:
    • Integer part (A) = 2
    • Numerator (B) = 1
    • Denominator (C) = 3
  2. Multiply the integer part by the denominator:
    • $A \times C = 2 \times 3 = 6$
  3. Add the numerator to the result:
    • $N = 6 + B = 6 + 1 = 7$
  4. We write down the result:
    • New numerator $N=7$ . Denominator $C=3$ (remains unchanged).
    • Answer: $\frac{7}{3}$

Example 2: Convert $5 \frac{3}{4}$

  1. We define the components: $A=5, B=3, C=4$.
  2. Multiply: $A \times C = 5 \times 4 = 20$.
  3. We add: $N = 20 + B = 20 + 3 = 23$.
  4. Answer: $\frac{23}{4}$.
