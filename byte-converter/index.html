KB vs KiB: The main confusion in the world of bytes

The main problem our calculator solves is the difference between how marketers sell us drives and how computers actually calculate capacity.

Decimal system (SI) - standard "1000"

Manufacturers of hard drives, SSDs and flash drives use decimal system (SI). It is convenient for marketing because it is based on powers of 10.

  • 1 Kilobyte (KB) = 1000 Bytes ( $10^3$ )
  • 1 Megabyte (MB) = 1000 Kilobytes = 1,000,000 Bytes ( $10^6$ )
  • 1 Gigabyte (GB) = 1000 Megabytes = 1,000,000,000 Bytes ( $10^9$ )

Binary system (IEC) - "1024" standard

Computers and their operating systems (like Windows) think binary terms (based on degree 2). There is a standard for this IEC.

  • 1 Kibibyte (KiB) = 1024 Bytes ( $2^{10}$ )
  • 1 Mebibyte (MiB) = 1024 Kibibytes = 1,048,576 Bytes ( $2^{20}$ )
  • 1 Gibibyte (GiB) = 1024 Mebibyte = 1,073,741,824 Bytes ( $2^{30}$ )

What's the problem? When you buy a 1TB (SI) drive, you get 1,000,000,000,000 Bytes. But your Windows OS (which counts by IEC, but signs it incorrectly) divides this number by 1024 three times (not 1000) to show you gigabytes:

$$\frac{1,000,000,000,000 \text{ Байт}}{1024 \times 1024 \times 1024} \approx 931.3 \text{ GiB (Гибибайт)}$$

Windows mistakenly calls this "931 GB", creating confusion. Our calculator allows you to choose a standard and see honest numbers.

Translation formulas

Our calculator uses these basic formulas for all calculations:

  • Bits to Bytes: $$\text{Байты} = \frac{\text{Биты}}{8}$$
  • Bytes to Bits: $$\text{Биты} = \text{Байты} \times 8$$
  • Bytes to Megabytes (MB, SI): $$\text{MB} = \frac{\text{Байты}}{1,000,000}$$
  • Bytes to Mebibytes (MiB, IEC): $$\text{MiB} = \frac{\text{Байты}}{1024 \times 1024}$$

Bits and Bytes: Internet Speed ​​Confusion

Important to remember:

  • Bytes (B) (with a capital "B") are used to measure file sizes And disk capacity.
  • Bits (b) (with a small "b") are used to measure data transfer rates (Internet, Wi-Fi).

When your provider promises 100 Mbit/s (Megabits per second), this means 100,000,000 bits per second. To find out at what speed the file will be downloaded, you need to convert this to Megabytes per second:

$$\frac{100,000,000 \text{ бит/с}}{8} = 12,500,000 \text{ Байт/с} = 12.5 \text{ МБ/с}$$

Thus, a 100 MB file will download not 1 second, but 8 seconds (100 MB / 12.5 MB/s).
