SMD resistor marking calculator (online decoding)

How to find out the value of an SMD resistor?

SMD (Surface Mount Device) resistors are miniature components used in modern electronics. Due to their small size, it is impossible to write the full resistance value on them, so special coding systems are used.

Our online calculator will help you instantly decipher any code printed on the resistor body. We support all existing labeling standards.

Types of SMD resistor markings

There are four main ways to encode denominations, and our tool works with each of them:

1. 3-digit marking

Used for resistors with standard tolerance (typically ±5%).

  • First two digits is the value (mantissa).
  • Third digit — multiplier (power of ten).

Example: 103

$$10 \times 10^3 = 10\,000 \text{ Ом} = 10 \text{ кОм}$$

2. 4-digit marking

Used for precision resistors with a tolerance of ±1%.

  • First three digits - meaning.
  • Fourth digit — multiplier.

Example: 4702

$$470 \times 10^2 = 47\,000 \text{ Ом} = 47 \text{ кОм}$$

3. Marking with the letter “R”

If the resistor value is very small (usually less than 10 ohms), the letter is used to represent the decimal point R.

Examples:

  • 4R7= $4.7 \text{ Ом}$
  • R22= $0.22 \text{ Ом}$
  • 0R22= $0.22 \text{ Ом}$

4. EIA-96 standard

This is the most difficult to read system used for 1% tolerance resistors. The code consists of two numbers and one letter (For example,01Cor68X).

  • Numbers (01-96) is a code corresponding to a specific denomination from Table E96.
  • Letter is a multiplier.

Example: 01C

  • 01according to the table corresponds to the value100.
  • Ccorresponds to the multiplier $100$ .
  • Result: $100 \times 100 = 10\,000 \text{ Ом} = 10 \text{ кОм}$ .

Our calculator automatically detects the type of code entered and returns the correct result, eliminating the need to manually search for EIA-96 tables.
