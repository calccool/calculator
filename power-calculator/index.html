Online calculator for raising a number to a power

Exponentiation calculator allows you to instantly calculate the value of an expression aⁿ, Where a - the base, and n - exponent. The tool supports negative, zero, and fractional powers, and automatically formats the result with the required precision.

Exponentiation formula:

aⁿ = a × a × a × ... (n раз)

For example, for a = 2 and n = 3 the result will be:

2³ = 2 × 2 × 2 = 8
