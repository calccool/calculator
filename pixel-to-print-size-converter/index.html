Print Size Calculator: Pixels, cm, Inches and DPI

Convert pixels to print size (cm, inches)

Before printing a photo or design, it is important to understand what size the physical copy will be without losing quality. This calculator links three key digital imaging parameters:

  1. Resolution in pixels (px): Actual digital file size.
  2. Physical Size (cm, mm, inches): Size of a sheet of paper.
  3. DPI / PPI (Dots Per Inch): Pixel density. Determines the clarity of the image.

How to use?

The tool works in two directions:

  1. Find out the print size: If you have a photo (for example, 1920x1080 px), you can find out which sheet it will fit on at a given quality (for example, 300 DPI).
  2. Find out the required pixels: If you need to print an A3 poster (29.7x42 cm) in good quality, the calculator will tell you what file resolution is required for this.

What is DPI and which one should you choose?

DPI (dots per inch) is a standard for measuring print quality.

  • 72 DPI: Standard for monitor screens. Not suitable for printing (the picture will be “pixelated” and blurry).
  • 150 DPI: Acceptable quality. Suitable for large posters that are viewed from a distance, or for newspaper printing.
  • 300 DPI: Gold standard for quality printing. Used for photographs, glossy magazines, business cards and booklets.
  • 600+ DPI: Professional ultra high definition fine art print.
