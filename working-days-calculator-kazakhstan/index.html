Formula and calculation method

The mathematical logic of the calculator is based on a sequential analysis of each day in a given range. The total number of working days ( $D_{work}$ ) is calculated using the following algorithm:

$$D_{work} = D_{total} — (D_{weekend} + D_{holidays} + D_{transfers})$$

Where:

  • $D_{total}$ — total number of calendar days in the period;
  • $D_{weekend}$ - number of Saturdays and Sundays (depending on the selected schedule 5/6);
  • $D_{holidays}$ - public holidays;
  • $D_{transfers}$ - additional days of rest resulting from the coincidence of a holiday and a day off.
