Theory and formulas for operations on sets

To understand the calculation results, use the reference below. The formulas use generally accepted mathematical notation.

1. Union

The union of the sets $A$ and $B$ is a set containing all elements that belong to at least one of these sets.

  • Formula: $A \cup B = \{x \mid x \in A \lor x \in B\}$
  • Example: If $A = \{1, 2\}$ and $B = \{2, 3\}$ , then $A \cup B = \{1, 2, 3\}$ .

2. Intersection

A set intersection is a set of elements that are present in both $A$ and $B$ .

  • Formula: $A \cap B = \{x \mid x \in A \land x \in B\}$
  • Example: The common element between $\{1, 2\}$ and $\{2, 3\}$ is the number $2$ .

3. Difference of sets

  • Difference A\B: Items that are present in $A$ but not in $B$ .
    • Formula: $A \setminus B$
  • Difference B\A: Items that are present in $B$ but not in $A$ .
    • Formula: $B \setminus A$

4. Symmetric Difference

This is a set of elements that belong to exactly one of two sets (either only $A$ or only $B$ , but not both). This is an analogue of logical “exclusive OR” (XOR).

  • Formula: $A \Delta B = (A \setminus B) \cup (B \setminus A)$

Application in programming (SQL and Python)

Understanding these operations is critical to working with databases and programming languages:

  1. SQL FULL OUTER JOIN — an analogue of the union ( $A \cup B$ ).
  2. SQL INNER JOIN — analogue of intersection ( $A \cap B$ ).
  3. Python Sets: Methods.union(), .intersection(), .difference()perform the same functions as our calculator.
