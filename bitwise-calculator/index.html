Online bit operations calculator: AND, OR, XOR, NOT and shifts

Bitwise operations are fundamental programming tools that allow data to be manipulated at the lowest level—the level of individual bits (0s and 1s). Unlike standard arithmetic, which operates on integers, bitwise arithmetic processes each digit of a number independently or shifts them relative to each other.

These operations are critical in systems programming, driver development, cryptography, and algorithm optimization. Our calculator allows you not only to get the result, but also to clearly see the process of changing each bit through an interactive bit matrix.

Basic logical operations

To understand how the calculator works, it is necessary to consider the truth table for basic functions. Each operation compares the corresponding bits of two operands (A and B) and produces a result (R).

Mathematical formulas for bitwise calculations

To formally describe processes, the following logical functions are used, where $x_i$ and $y_i$ are the corresponding operand bits at position $i$ :

  • Bitwise AND: $f(x, y) = x \land y$
  • Bitwise OR: $f(x, y) = x \lor y$
  • Exclusive OR: $f(x, y) = x \oplus y$
  • Inversion: $f(x) = \neg x$

Bit shifts: SHL and SHR

Bit shifts move all the digits of a number to the left or right by a specified number of positions. This is equivalent to quickly multiplying or dividing by powers of two.

Shift left (SHL - Shift Left)

When shifting left, the bits are moved towards the most significant bits, and the vacated positions on the right are filled with zeros.

Formula for shifting the number $n$ to $s$ positions:

$$Result = n \cdot 2^s$$

Shift Right (SHR - Shift Right)

When shifting to the right, the bits move towards the least significant bits. This corresponds to integer division.

Formula for shifting the number $n$ to $s$ positions:

$$Result = \lfloor n / 2^s \rfloor$$
