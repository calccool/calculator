SLU 2×2 Calculator: Solve System of Linear Equations using Cramer Method

Solving a system of linear equations 2×2 using Cramer's method

A system of linear equations (SLE) is a set of $m$ equations with $n$ unknowns, where each equation is linear. Our online calculator is designed to quickly and accurately solve systems of size 2×2 (two variables and two equations), using classic and reliable Cramer method.

Using a calculator allows you not only to get the answer, but also to see all the intermediate steps, including the calculation of determinants, which is extremely important for educational purposes and testing independent work.

General view of the 2×2 system

Any system of linear equations with two variables ( $x$ and $y$ ) can be represented in the following general form, where $a_i$ , $b_i$ are the coefficients of the variables, and $c_i$ are the free terms:

$$\begin{cases} a_1x + b_1y = c_1 \\ a_2x + b_2y = c_2 \end{cases}$$

How the Cramer Method works

Cramer's method is based on the calculation of three main determinants, which allows you to quickly and unambiguously find a solution to the system (provided that the main determinant is not equal to zero).

1. Main system identifier ( $\Delta$ )

The main determinant is composed of coefficients for the variables ( $x$ and $y$ ):

$$\Delta = \begin{vmatrix} a_1 & b_1 \\ a_2 & b_2 \end{vmatrix} = a_1b_2 — a_2b_1$$

2. Determinant by variable $x$ ( $\Delta_x$ )

The determinant of $\Delta_x$ is obtained by replacing the first column (coefficients $a_1$ , $a_2$ ) with a column of free terms ( $c_1$ , $c_2$ ):

$$\Delta_x = \begin{vmatrix} c_1 & b_1 \\ c_2 & b_2 \end{vmatrix} = c_1b_2 — c_2b_1$$

3. Determinant by variable $y$ ( $\Delta_y$ )

The determinant of $\Delta_y$ is obtained by replacing the second column (coefficients $b_1$ , $b_2$ ) with a column of free terms ( $c_1$ , $c_2$ ):

$$\Delta_y = \begin{vmatrix} a_1 & c_1 \\ a_2 & c_2 \end{vmatrix} = a_1c_2 — a_2c_1$$

Formulas for finding a solution

If the main determinant $\Delta \ne 0$ (is not equal to zero), then the system has a unique solution, which is found according to the formulas:

$$x = \frac{\Delta_x}{\Delta}$$

$$y = \frac{\Delta_y}{\Delta}$$

The most important advantage of the Cramer Method is the ability to quickly determine the type of solution of a system based on the values ​​of the determinants.
