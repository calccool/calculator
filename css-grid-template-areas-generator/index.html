Mathematical logic and mesh calculations

When calculating the grid, the calculator takes into account the relationship between the number of columns, gaps and the final width of the container. To calculate the total width or height of a grid with fixed values, use the following formula:

$$L_{total} = \sum_{i=1}^{n} size_i + (n — 1) \cdot gap$$

Where:

  • $L_{total}$ — final length (width or height);
  • $n$ - number of columns or rows;
  • $size_i$ - specific cell size;
  • $gap$ - intergrid space size.

If the unit of measurement is used1fr(fractional unit), the calculation of the available space for each $W_{fr}$ column occurs according to the principle:

$$W_{fr} = \frac{W_{container} — (n — 1) \cdot gap}{\sum fr}$$
