Mathematical model of URL formation

The link generation process is based on a strict string concatenation algorithm. The final URL can be represented as the following formula:

$$URL_{final} = URL_{base} + \text{S} + \sum_{i=1}^{n} (P_i + V_i + \text{D})$$

Where:

  • $URL_{base}$ is the source address of your landing page.
  • $\text{S}$ - separator (character?for the first parameter or&, if the link already has parameters).
  • $P_i$ - UTM parameter name (for example,utm_source).
  • $V_i$ - parameter value passed through filters (lower case and transliteration).
  • $\text{D}$ — separator between pairs of parameters (&).
