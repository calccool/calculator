Subset and Boolean calculator: quantity calculation and list generation

This online calculator is designed to work with set theory. It allows you not only to determine the total number of possible subsets, but also to visualize them, construct a Boolean (the set of all subsets) and find combinations of a certain length.

How is the number of subsets calculated?

In combinatorics, the number of all possible subsets for a finite set consisting of $n$ elements is equal to two to the power of $n$ .

The formula looks like this:

$$N = 2^n$$

Where:

  • N — total number of subsets.
  • n — the power of the set (the number of elements in it).

This set of all subsets is called Boolean (Power Set). It is important to remember that a Boolean always includes the original set itself and the empty set ( $\emptyset$ ).

Own subsets

Often in problems you need to find the quantity own (nontrivial) subsets. These are all subsets except the original set itself.

Formula for eigensubsets:

$$N_{proper} = 2^n — 1$$

Subsets of a given size (k)

If you want to know how many ways there are to select exactly $k$ elements from a set of size $n$ (without regard to order), use the formula for the number of combinations (binomial coefficient):

$$C_n^k = \frac{n!}{k!(n-k)!}$$

Our calculator automatically performs this calculation if you specify the parameter k.
